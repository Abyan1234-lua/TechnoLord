# Title Tag & Meta Description Templates

*Sourced from: The Art of SEO — Enge, Spencer, Fishkin & Stricchiola (Ch. 6: Developing an SEO-Friendly Website)*

---

## Title Tag Character Counter Reference

| Length | Status |
|---|---|
| Under 50 characters | Acceptable — may be leaving ranking value on table |
| 50–65 characters | **Optimal zone** |
| 66–70 characters | Borderline — Google sometimes shows, sometimes truncates |
| Over 70 characters | Truncated in SERPs; excess words receive less weight |

**Counting tip:** Include spaces in your count. "Cordless Drill Buying Guide | Acme Tools" = 40 characters.

---

## Title Tag Templates by Page Type

### E-Commerce

```
Product page (short name):
{Product Name} — {Brand}
Example: "DeWalt 20V MAX Cordless Drill — Acme Tools"  [41 chars]

Product page (with attribute):
{Product Name} {Key Attribute} — {Brand}
Example: "Makita 18V Brushless Drill Kit — Acme Tools"  [44 chars]

Category page:
{Category Keyword} — {Brand}
Example: "Cordless Drills — Acme Tools"  [29 chars]

Category with modifier:
{Primary Keyword}: {Secondary Keyword} — {Brand}
Example: "Cordless Drills: Professional Grade — Acme Tools"  [49 chars]
```

### Content / Blog

```
How-to post:
How to {Action} — {Brand Blog}
Example: "How to Choose a Cordless Drill — Acme Blog"  [43 chars]

List post:
{Number} Best {Category} for {Use Case} — {Brand}
Example: "7 Best Cordless Drills for Home Use — Acme Tools"  [48 chars]

Guide / pillar:
{Primary Keyword}: Complete Guide — {Brand}
Example: "Cordless Drills: The Complete Buying Guide — Acme"  [49 chars]

Question-based:
What Is {Topic}? {Qualifier} — {Brand}
Example: "What Is a Brushless Motor? Drill Guide — Acme"  [46 chars]
```

### Local Business

```
Homepage:
{Brand} — {Service} in {City}, {State}
Example: "Acme Tools — Power Tools in Austin, TX"  [38 chars]

Service page:
{Service} in {City} — {Brand}
Example: "Cordless Drill Repair in Austin — Acme Tools"  [44 chars]

Location-specific category:
{Category} Near {City} — {Brand}
Example: "Power Tools Near Downtown Austin — Acme Tools"  [45 chars]
```

### SaaS / Service

```
Homepage:
{Brand} — {Primary Value Proposition}
Example: "Acme Analytics — Real-Time Sales Reporting"  [43 chars]

Feature page:
{Feature Name}: {Benefit} — {Brand}
Example: "Sales Dashboard: Track Revenue in Real Time — Acme"  [49 chars]

Comparison page:
{Brand} vs {Competitor} — Full Comparison
Example: "Acme Analytics vs Salesforce — Full Comparison"  [46 chars]
```

## Title Tag Divider Reference

| Divider | Use case | Example |
|---|---|---|
| `\|` (pipe) | Most common; clean separation | `Cordless Drills \| Acme Tools` |
| `-` (dash) | Works well; slightly less formal | `Cordless Drills - Acme Tools` |
| `:` (colon) | For "category: subcategory" structures | `Drills: Cordless & Impact` |
| `>` (arrow) | Implies hierarchy | `Tools > Drills > Cordless` |
| `—` (em dash) | Premium feel | `Cordless Drills — Acme Tools` |

**No divider needed** when the title naturally reads as a complete phrase: `"How to Choose a Cordless Drill"` — no brand needed on blog posts with a standalone title.

## Meta Description Templates

### Character limit
**Target: 150–155 characters (with spaces)**
Google displays up to 160; truncation at 155 is the safe ceiling.

### Structure formula

```
[What the page is] + [specific benefit or value] + [call to action]
```

### Templates by page type

**Product page:**
```
Shop {product name} at {brand}. {key feature or benefit}. {CTA + differentiator}.
Example: "Shop DeWalt cordless drills at Acme Tools. 20V MAX battery, 5-year warranty.
Free shipping on orders over $99."
[Character count: 118 ✓]
```

**Category page:**
```
Browse our range of {category} — {descriptor}. {proof point or range}. {CTA}.
Example: "Browse our range of cordless drills — professional grade to DIY. 50+ models
from DeWalt, Makita & Milwaukee. Fast delivery."
[Character count: 121 ✓]
```

**Blog post / guide:**
```
{What this article covers} + {who it's for} + {key takeaway or benefit}.
Example: "Learn how to choose the right cordless drill for your project. Our guide
covers voltage, chuck size, and battery life for every budget."
[Character count: 133 ✓]
```

**Homepage:**
```
{Brand name}: {primary value proposition}. {proof point}. {CTA}.
Example: "Acme Tools: Professional power tools for contractors and serious DIYers.
Over 10,000 products in stock. Shop online or visit us in Austin."
[Character count: 140 ✓]
```

**Local service page:**
```
{Service} in {city} from {brand}. {differentiator}. {social proof or trust signal}. {CTA}.
Example: "Cordless drill repair in Austin from Acme Tools. Same-day service available.
Trusted by contractors since 1998. Book online."
[Character count: 121 ✓]
```

## Common Title Tag Errors

| Error | Example | Fix |
|---|---|---|
| Brand name leads, keyword trails | `"Acme Tools — Cordless Drills"` | `"Cordless Drills — Acme Tools"` |
| Over 65 characters | `"The Ultimate Comprehensive Guide to Buying Cordless Power Drills for Home and Professional Use"` | `"Cordless Drill Buying Guide: Home & Pro — Acme"` |
| No keyword in title | `"Products Page 4"` | `"Cordless Drills: Shop All Models — Acme Tools"` |
| Duplicate across pages | Same title on every category page | Use per-page formula with unique category name |
| Keyword stuffed | `"Best Cordless Drill Cordless Drills Drill Cordless Buy"` | `"Best Cordless Drills — Shop All Brands — Acme"` |
| Generic / not descriptive | `"Home"` `"Products"` `"About"` | `"Power Tools & Equipment — Acme Tools"` |
| Wrong intent signal | `"DeWalt 20V Drill Information"` (on a product/buy page) | `"Buy DeWalt 20V MAX Drill — Best Price — Acme"` |

## When to Skip the Meta Description

Write a custom meta description for:
- High-traffic landing pages
- Pages targeting 1–3 specific, high-value keyword phrases
- Any page where auto-extracted text is consistently poor or incomplete

Skip the meta description (let the engine extract) for:
- Blog archives with hundreds of posts
- Large product catalogs (thousands of SKUs)
- Pages with diverse long-tail traffic where forcing one description would reduce relevance for most queries
- Any page where writing unique quality descriptions is not operationally feasible

**Key insight:** When a search engine extracts text automatically, it always displays the words that match the user's actual search query. A forced meta description that doesn't contain the search term reduces visual relevance in the SERP.
