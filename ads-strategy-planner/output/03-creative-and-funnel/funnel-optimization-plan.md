# Funnel Optimization Plan

## Objective

Improve conversion efficiency by balancing friction, qualification, and buyer readiness.

## Funnel Principles

- Friction is adjustable, not fixed.
- Add friction when lead quality is poor.
- Remove friction when volume is constrained.
- Use proof assets to reduce trust barriers at higher spend levels.

## Recommended Funnel Structure

1. **Ad Click**
2. **Intent-Aligned Landing Page** (clear promise + proof + CTA)
3. **Qualification Layer** (form depth or pre-qualifying question set)
4. **Conversion Event** (call booking, lead form, checkout)
5. **Follow-up Sequence** (email/SMS/remarketing)

## Friction Control Matrix

| Signal | Action |
|---|---|
| High volume, low quality | Add qualification steps, stronger intent filters |
| Low volume, high quality | Reduce fields, shorten path to conversion |
| Drop-off at first step | Improve message match and first-screen clarity |
| Drop-off after form start | Simplify form and clarify value exchange |

## Shadow Funnel (Retargeting Layer)

- Install and validate tracking pixels across active channels.
- Build retargeting audiences from non-converted paid traffic.
- Serve proof-first and objection-handling creative sequences.

## Weekly Optimization Checklist

- Review step-level conversion rates.
- Identify highest drop-off points.
- Run one friction adjustment test per week.
- Sync sales feedback on lead quality with paid media changes.
